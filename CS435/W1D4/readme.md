## Question 4 — Fractional Knapsack, W_max = 20

Same items as Q3. Sort by value-to-weight ratio (descending), then fill the knapsack from the top.

| Item | Value v | Weight w | v / w | Order |
|:----:|--------:|---------:|------:|:-----:|
| d | 16 | 2 | 8.00 | 1 |
| a | 25 | 5 | 5.00 | 2 |
| e | 28 | 7 | 4.00 | 3 |
| c | 24 | 8 | 3.00 | 4 |
| b | 12 | 6 | 2.00 | 5 |

Greedy fill:

- Take all of **d**  (w = 2)  → used 2 / 20,  value = 16
- Take all of **a**  (w = 5)  → used 7 / 20,  value = 16 + 25 = 41
- Take all of **e**  (w = 7)  → used 14 / 20, value = 41 + 28 = 69
- Only 6 of 8 weight units left for **c**  →  take 6/8 = 0.75 of c  →  value = 0.75 × 24 = 18
- Knapsack full at 20 / 20 — skip **b**

Optimal fractional value = 16 + 25 + 28 + 18 = **87**.

Solution: take all of d, a, e and three-quarters of c.

## Question 5 — Climbing Stairs (LeetCode 70)

```java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int prev2 = 1, prev1 = 2;
    for (int i = 3; i <= n; i++) {
        int curr = prev1 + prev2;
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```

## Question 6 — House Robber (LeetCode 198)

```java
public int rob(int[] nums) {
    int prev2 = 0, prev1 = 0;
    for (int n : nums) {
        int curr = Math.max(prev1, prev2 + n);
        prev2 = prev1;
        prev1 = curr;
    }
    return prev1;
}
```
